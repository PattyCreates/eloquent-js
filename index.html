Values

Binary code is just data. Values are representation of that data. There are six basic types of values: numbers, strings, Booleans, objects, functions and undefined values.

Numbers and arithmetic

100 + 4 * 12

  • + * / - symbols are called operators.
  • * / have higher precedence so will be executed before + -.
  • If they have similar presidence, it executes left to right.
  • A modulo operator is a 140 % 30 where it divides and then gives the remainer. So, it will return a value of the remainder, 20.

Strings

Text.

  • It can be wrapped in single or double quotes.
  • It can be "con" + "cat" + "enated" + "."
  • Escape quotes using a \.
  • /n makes a new line.
  • /t adds a tab.

Boolean Values, Comparisons, and Boolean Logic

Comparisons

  • Only two values of Boolean. true and false.
  • Alphabetical holds true. "Bertha" < "Charlie"
  • But, capitalization holds true as well. "bertha" > "Charlie". So Uppercase is less than lowercase. Which is kinda contrarian. But it holds up to the unicode standard.
  • Comparisons are left to right.
  • true && false returns false. && needs to have both be true. so true && true does work.
  • || is or so either value has to return true. true || false will return true.
  • Here are few more operators.
    • ! is not.
    • != is not equal to. !true comes out false
    • == is equal to.
    • === is absolutely not equal to.
    • >= is greater than or equal to.
    • =< is equal to or less than.

Variables

Variables store data. Can be altered. Not permanent. Sometimes they can be holding expressions which are simple data values such as numbers or strings (they don't perform anything, they just are). On other times they can be holding statements, which are functions with outputs. Statements end with a ;. Variables can hold either.

var berthasDebt = 150;

var berthasDebt = 150-20;

//berthasDebt will return to be 130.
